# YouTube-Gadgets

YouTube Gadgets is a Spring Boot + Thymeleaf based web application that helps users analyze YouTube videos quickly and efficiently. By simply providing a YouTube video URL, the app can:
✨ Features
- 🔍 SEO Tags Finder – Extracts SEO metadata (title, description, keywords) from the given YouTube video URL.
- 🖼️ Thumbnail Generator – Fetches and displays the video thumbnail instantly.
- 🌐 Web UI with Thymeleaf – Clean and dynamic HTML templates powered by Thymeleaf.
- 🎨 Frontend Styling – Responsive design using CSS and JavaScript for better user experience.
- ⚡ Spring Boot Backend – Robust backend architecture for handling requests and integrating YouTube data.
