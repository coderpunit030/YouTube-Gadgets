# YouTube-Gadgets

YouTube Gadgets is a smart toolkit that helps users explore YouTube more effectively. With just a URL provided by the user, it can:
✨ Features
- 🔍 SEO Tags Finder – Extracts and displays SEO tags (title, description, keywords, etc.) from the given YouTube video URL.
- 🖼️ Thumbnail Generator – Instantly fetches and shows the video thumbnail from the provided URL.
- 📊 Video Insights (optional) – Can be extended to show metadata like views, likes, and publish date.
- ⚡ Simple & Fast – Designed for quick results with minimal input.
